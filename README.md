# Mizudo-app
Aplicação para gerenciamento de membros da associação Mizu-do de Karatê.
  
## Tecnologias utilizadas:
 - Java 18
 - Quarkus
  - JAX-RS
  - Autenticação com JWT
  - JPA
 - Angular 13
 - *Ionic v6 (em estudos)  
 - PostgreSQL

## Hospedagem
  - Heroku
